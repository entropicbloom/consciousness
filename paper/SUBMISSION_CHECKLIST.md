# Submission Checklist

Use this checklist to track your submission preparation.

## Pre-Submission Tasks

### Author Information
- [ ] Update author name in `main.tex` (line 36)
- [ ] Update author affiliation in `main.tex` (line 41)
- [ ] Update author email in `main.tex` (lines 41, 44)
- [ ] Update author name in `main-article.tex` (line 33)
- [ ] Update author affiliation in `main-article.tex` (line 34)
- [ ] Update author email in `main-article.tex` (line 35)

### Cover Letter
- [ ] Open `COVER_LETTER.txt`
- [ ] Add your name at the bottom
- [ ] Add your affiliation at the bottom
- [ ] Add your email at the bottom
- [ ] Customize letter if desired (optional)

### Paper Review
- [ ] Read through entire paper
- [ ] Check abstract (max 200 words) - currently good
- [ ] Verify keywords are appropriate
- [ ] Check all figure references
- [ ] Check all citations
- [ ] Spell check
- [ ] Grammar check

### Optional but Recommended
- [ ] Identify 3-5 potential reviewers
  - [ ] Name, affiliation, email, expertise for each
  - [ ] Ensure no conflicts of interest
- [ ] Compile paper locally (if you have LaTeX) to check for errors
- [ ] Review figures to ensure they're high quality

## MDPI Account Setup
- [ ] Create MDPI account at https://www.mdpi.com/about/account
- [ ] Verify email address
- [ ] Complete profile (optional)

## Submission Process

### Navigate to Special Issue
- [ ] Go to https://www.mdpi.com/journal/entropy/special_issues/5VXL7P10J1
- [ ] Click "Submit to Special Issue" button
- [ ] Confirm you're logged in to MDPI

### Upload Files
- [ ] Upload main manuscript file
  - Choose: `main-article.tex` (easier) OR `main.tex` (MDPI template)
  - Alternative: Upload compiled PDF
- [ ] Upload all figures from `../figures/` directory (18 files)
  - Files can be uploaded individually or as zip
  - Or embed in PDF if submitting compiled version
- [ ] Upload `references.bib` (if submitting LaTeX source)

### Enter Metadata
- [ ] Title: "Unambiguous Representations in Neural Networks: A Relational Structure Approach to Consciousness"
- [ ] Abstract: Copy from paper
- [ ] Keywords: consciousness; neural correlates; representation; ambiguity; information theory; neural networks; relational structure
- [ ] Author information:
  - [ ] Full name(s)
  - [ ] Affiliation(s)
  - [ ] Email(s)
  - [ ] ORCID ID(s) if available
- [ ] Corresponding author marked

### Article Details
- [ ] Article type: **Research Article**
- [ ] Special issue: Information-Theoretic Methods in Computational Neuroscience
- [ ] Section: Information Theory, Probability and Statistics

### Cover Letter
- [ ] Paste cover letter from `COVER_LETTER.txt`
- [ ] Verify all placeholders are replaced
- [ ] Check formatting

### Suggested Reviewers (Optional)
- [ ] Reviewer 1: Name, affiliation, email, expertise
- [ ] Reviewer 2: Name, affiliation, email, expertise
- [ ] Reviewer 3: Name, affiliation, email, expertise
- [ ] Reviewer 4: Name, affiliation, email, expertise (optional)
- [ ] Reviewer 5: Name, affiliation, email, expertise (optional)

### Final Checks
- [ ] Review all entered information
- [ ] Check for typos in metadata
- [ ] Verify email addresses are correct
- [ ] Ensure all files uploaded successfully
- [ ] Read submission terms and conditions
- [ ] Confirm originality and no dual submission

### Submit
- [ ] Click final "Submit" button
- [ ] Save confirmation email
- [ ] Note manuscript ID number
- [ ] Note submission date

## Post-Submission

### Immediate
- [ ] Check email for submission confirmation
- [ ] Save manuscript ID in safe place
- [ ] Add to calendar: Check status in 3 weeks

### During Review (~3 weeks)
- [ ] Monitor email for editorial correspondence
- [ ] Respond promptly to any requests
- [ ] Have additional data/analyses ready if needed

### If Revisions Requested
- [ ] Read reviewer comments carefully
- [ ] Prepare point-by-point response
- [ ] Make requested changes
- [ ] Track all changes for resubmission
- [ ] Submit within deadline (usually 2-4 weeks)

### If Accepted
- [ ] Pay Article Processing Charge (CHF 2600)
- [ ] Review proofs carefully
- [ ] Provide corrections if needed
- [ ] Receive DOI and publication link
- [ ] Share published paper!
